English version

grammar ディレクトリ

型階層，スキーマ，プリンシプルなどが定義されたファイルが含まれます．こ れらのファイルは $PREFIX/share/liblilfes/enju/ にインストールされます．

display.lil
sign などを表示用文字列に変換するプログラム
forestevent.lil
feature forest model のイベントを取りだすプログラム
forestprob.lil
feature forest model の確率値を計算するプログラム
fs.css
素性構造をブラウザで見るためのスタイルシート
fs.xsl
素性構造をブラウザで見るための XSL
grammar.lil
up のインタフェースを実装しているモジュール
lexmask.lil
lexicalized unigram FOM モデルのマスク
lexmodel.lil
lexicalized unigram FOM モデルを実装しているモジュール
macro.lil
sign のいろいろなマクロを実装しているモジュール
moriv.css
moriv で使うスタイルシート
moriv.lil
moriv で構文解析結果を見るためのプログラム
outputdep.lil
構文解析結果を dependency 形式で出力するプログラム
outputxml.lil
構文解析結果を xml 形式で出力するプログラム
preproc.lil
単語と品詞の前処理(stemming 以外)を行なうプログラム
schema.lil
ID schema と principle を実装しているモジュール
semantics.lil
semantic principle を実装しているモジュール
stem.lil
stemming のプログラム
synmask.lil
syntax FOM モデルのマスク
synmodel.lil
syntax FOM モデルを実装しているモジュール
tools.lil
パーザでよく使う述語集
types.lil
HPSG 文法の型階層を定義しているモジュール
unievent.lil
unigram model のイベントを取りだすプログラム
uniprob.lil
unigram model の確率値を計算するプログラム

tools ディレクトリ

いろいろなスクリプトが含まれます．

err2html.prl
*.err ファイルを HTML 形式に変換する Perl スクリプト
mrg2trees.prl
Penn の *.mrg ファイルを *.trees 形式に変換する Perl スクリプト
trees2raw.prl
*.trees ファイルを生テキストに変換する Perl スクリプト
trees2tagged.prl
*.trees ファイルを POS タグつきコーパスに変換する Perl スクリプト
