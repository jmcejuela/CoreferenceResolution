インストール方法

English version

MAYZ ツールキットは Enju パッケージの一部として提供されています． MAYZ/Enju のコンパイルにはLiLFeSを使いま すので，これがインストールされている必要があります．また，g++ のバージョ ン 3.2 以上が必要です．

  1. ソースパッケージをダウンロードして，展開します． 
    % tar xvzf enju-X.Y.Z.tar.gz
  2. configure スクリプトを実行します． 
    % ./configure
  3. make します． 
    % make
  4. mayz-src/ の下にコマンドが生成されます．

CVS からパッケージをとった人へ

はじめに，configure スクリプトを作るために以下を実行する必要があります． あとは上と同じです． 
% aclocal; autoheader; automake --add-missing --foreign; autoconf
